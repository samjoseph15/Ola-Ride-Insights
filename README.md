 __Ola-Ride-Insights__

__Project Overview__

**Ola Ride Insights** is a ride-sharing analytics project focused on cleaning data, performing EDA, writing SQL queries, building Power BI dashboards, and developing a Streamlit app.

__Objective__
The project aims to analyze Ola ride data and generate insights on booking trends, cancellations, revenue, and ratings.

__Skills Gained__
- SQL querying
- Data cleaning and preprocessing
- Exploratory Data Analysis
- Power BI visualization
- Streamlit app development
- Business intelligence insights

__Business Use Cases__
- Identifying peak demand hours.
- Analyzing customer behavior.
- Understanding pricing patterns.
- Detecting cancellation trends.
- Evaluating customer and driver ratings.

__Approach__

__Data Understanding and Exploration__
Understand the dataset structure and key variables such as ride status, payment method, and ratings.

__Data Cleaning and Preprocessing__
Handle missing values, standardize formats, and convert data types.

__SQL Query Development__
Write queries to extract insights on ride trends, cancellations, ratings, and revenue.

__Power BI Dashboard Creation__
Create interactive visualizations for ride volume, booking status, cancellations, revenue, and ratings.

__Streamlit Application Development__
Build a user-friendly app to display SQL results and dashboard insights.

__Key SQL Questions__
- Successful bookings
- Average ride distance by vehicle type
- Cancelled rides by customers
- Top customers by ride count
- Driver cancellations due to personal or car issues
- Driver ratings for Prime Sedan bookings
- Rides paid using UPI
- Average customer rating by vehicle type
- Total booking value of successful rides
- Incomplete rides with reasons

__Power BI Visuals__
- Ride volume over time
- Booking status breakdown
- Top vehicle types by ride distance
- Average customer ratings by vehicle type
- Cancellation reasons
- Revenue by payment method
- Top customers by total booking value
- Ride distance distribution per day
- Driver ratings distribution
- Customer vs. driver ratings
